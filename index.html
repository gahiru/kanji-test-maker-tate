使い方
1. 共通情報と、左の漢字テスト / 右の読みテストを入力します。
2. 問題は [答え|テストに出す文字] の形で入力します。
3. 固定文字を分けたいときは // を使います。// は印刷されません。空白の代わりではなく、文節の区切り記号です。 例：パン//を[売る。|★うる。]
4. 読みテストは同じ10問から自動で作成します。ランダム並び替え後に保存すると、ファイル名の最後に _ランダム並び替え が付きます。
5. 縦版では、漢字だけの問題は四角のマス、★の問題は ( )、読みテストはすべて ( ) で表示します。
